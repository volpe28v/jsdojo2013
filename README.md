#jsdojo2013で作成したアンケートアプリ

## どんなアプリか
* 楽しく「ふりかえり」ながらアンケートができて、フィードバックも楽しめる(予定の)アプリです。
* アンケートを書く時に一人で小さな「ふりかえり」ができて、アンケート送信後にもみんなのアンケート結果に、部分的に「いいね！」や「いくない」ができます。
* 自分の書いたアンケートが他の人に共感されたかそうでないかを見に行くのが楽しくなりそうなアプリです！
